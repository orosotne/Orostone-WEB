# OROSTONE — Production Readiness Audit

**Auditor:** Senior staff engineer perspective (Apple-grade rigor)
**Scope:** Complete codebase, infra, Vercel config, Supabase schema, dist bundle
**Date:** 2026-05-07
**Branch:** `main` @ `c726249`

This document is committed as a reference for `/ultrareview` and future audits.
The auditor used direct grep/read tool calls because 8 parallel sub-agents repeatedly
failed with `Autocompact thrashing` due to extreme file sizes (`pages/ShopProductDetail.tsx`
130 KB, `pages/Shop.tsx` 63 KB, `pages/SinterovanyKamen.tsx` 48 KB, etc.).
**File size alone is a production red flag** — see P0-3, P1-29, P1-30, P1-31.

## SHIP-BLOCKERS (P0 — fix before going live or first business hour)

### 🔴 P0-1 — Supabase RLS data leak: PII všetkých zákazníkov a leadov verejne čitateľné

**Súbor:** `supabase/schema.sql:126-167`

**Dôkaz:**
```sql
CREATE POLICY "Allow anonymous select" ON customers       FOR SELECT USING (true);
CREATE POLICY "Allow anonymous select" ON quotes          FOR SELECT USING (true);
CREATE POLICY "Allow anonymous select" ON newsletter_subscribers FOR SELECT USING (true);
```

**Exploit:** `VITE_SUPABASE_ANON_KEY` je v každom .js bundli (verejný kľúč).
Komentár tvrdí "potrebné pre RETURNING po INSERT", **ale RETURNING SELECT policy
nepotrebuje** — postačuje `INSERT WITH CHECK`. Útočník urobí
`supabase.from('customers').select('*')` a má všetky mená, emaily, telefóny,
adresy zákazníkov a všetky leady vrátane projektových súborov.

**Fix:** Odstráň anonymné SELECT policies. Pre `RETURNING` pridaj scoped policy
`USING (id = current_setting('app.current_customer_id'))` alebo presuň
`findOrCreateCustomer` do Edge Function so service-role kľúčom. Rovnako uprav
`quotes` aj `newsletter_subscribers`. **Toto musí byť opravené pred hodinou ostrou.**

### 🔴 P0-2 — Vercel.json chýba Cache-Control pre /assets/*, /fonts/*, /images/*

**Súbor:** `vercel.json:69-88`

**Dôkaz:** Headers blok obsahuje len bezpečnostné hlavičky. Vite hashuje názvy
bundlov (`index-B_UgNZs4.js`), takže by mali byť `public, max-age=31536000, immutable`.
**Bez toho každý opakovaný návštevník stiahne 2 MB JS znovu.**

**Fix:**
```json
{
  "source": "/assets/(.*)",
  "headers": [{ "key": "Cache-Control", "value": "public, max-age=31536000, immutable" }]
},
{
  "source": "/fonts/(.*)",
  "headers": [{ "key": "Cache-Control", "value": "public, max-age=31536000, immutable" }]
},
{
  "source": "/images/(.*)",
  "headers": [{ "key": "Cache-Control", "value": "public, max-age=2592000" }]
}
```

### 🔴 P0-3 — BlogArticle chunk = 552 KB (najväčší chunk v appke)

**Súbor:** `pages/BlogArticle.tsx` (40 KB source) → `dist/assets/BlogArticle-C_3AMBu7.js = 552K`

**Dôkaz:** Všetkých 5+ blog článkov je s veľkou pravdepodobnosťou inline-ovaných
ako konštanty v jednom chunku. Užívateľ musí stiahnuť 552 KB len na jeden článok.

**Fix:** Presuň každý článok do samostatného `data/articles/<slug>.tsx` súboru
a `lazy()` import podľa slugu cez `useParams`. Cieľ: < 60 KB per route.

### 🔴 P0-4 — Main app bundle (index-*.js) = 516 KB

**Súbor:** `dist/assets/index-B_UgNZs4.js = 516K`

**Dôkaz:** Aj po manualChunks split je app bundle pol megabajtu. Pravdepodobne
`Shop.tsx` (63 KB source) + `EshopLayout` + `CartContext` + non-lazy importy ťahajú všetko.

**Fix:**
1. Spusti analyzer: `npm i -D rollup-plugin-visualizer` a pridaj plugin do
   `vite.config.ts`. Identifikuj top 10 modulov.
2. `pages/Shop.tsx` má 63 KB — rozdeľ na sekcie (`HeroSection`, `InspirationSection`,
   `ProductGrid`...) a `lazy()` to-below-the-fold sekcie.
3. Zváž odstránenie GSAP+ScrollTrigger (72 KB) ak ho používa len 1-2 sekcie —
   Framer Motion alebo CSS-only animácie postačia.

### 🔴 P0-5 — Build neobsahuje `tsc --noEmit` typecheck

**Súbor:** `package.json:8`

**Dôkaz:** `"build": "tsx scripts/sync-shop-fallback.ts && vite build && tsx scripts/prerender.ts"` —
Vite `transform` ignoruje TS errory, len ich strippuje. **TypeScript chyby
NEZASTAVIA deploy.**

**Fix:** `"build": "tsc --noEmit && tsx scripts/sync-shop-fallback.ts && vite build && tsx scripts/prerender.ts"`.
Pred-aj ošetri existujúce errory: `tsc --noEmit | head -50`.

### 🔴 P0-6 — Žiadne automatizované testy v produkčnom e-commerce

**Súbor:** `package.json` — žiadny `test` script, žiadny `vitest`/`jest`/`playwright` v deps.

**Riziko:** Jediná regresná ochrana je manuálne klikanie. Pre prémium e-shop
s objednávkami €500-€3000 to nestačí.

**Fix (minimum):**
1. Vitest + 1 happy-path test pre `CartContext` (add → remove → checkout intent).
2. Playwright e2e pre kritickú cestu: home → produkt → add to cart → checkout
   form fills → proceed-to-payment redirect.
3. CI gate: PR sa nemerge bez green CI.

### 🔴 P0-7 — Account.tsx, Login.tsx, Register.tsx = 1 229 riadkov mŕtveho kódu

**Súbory:** `pages/Account.tsx` (692 LOC), `pages/Login.tsx` (217 LOC), `pages/Register.tsx` (320 LOC)

**Dôkaz:** Nie sú routed v `EshopApp.tsx`. `vercel.json:38-41` redirectuje
`/login`, `/register`, `/ucet`, `/objednavky` priamo na Shopify Customer Accounts.
Tieto súbory sú v repo neskôr zatiaľ ako (potenciálne) tree-shake-vateľné.

**Fix:** Vymazať. Kompletne. Aj ich SEOHead absencia (zostávajúce 3 z 24 pages)
sa tým vyrieši.

### 🔴 P0-8 — Skip link chýba (WCAG 2.4.1 fail, EU EAA 2025 blocker)

**Dôkaz:** `grep "skip|preskočiť|main-content"` na celej kódbáze = 0 výsledkov.

**Fix:** Pridaj na vrch `EshopLayout.tsx`:
```tsx
<a href="#main" className="sr-only focus:not-sr-only focus:absolute focus:top-2 focus:left-2 focus:bg-brand-dark focus:text-white focus:px-4 focus:py-2 focus:rounded">
  Preskočiť na hlavný obsah
</a>
```
A `<main id="main">` wrapper.

### 🔴 P0-9 — `prefers-reduced-motion` rešpektuje len 4 miesta z celej kódbázy

**Dôkaz:** `grep useReducedMotion|prefers-reduced-motion` = 4 výsledky
(Shop.tsx:325, ProductCatalog.tsx:183, InspirationSection.tsx:471, 472).

**Riziko:** GSAP, Framer Motion, ScrollTrigger animácie idú naplno aj pri
systémovom "Reduce Motion" — vestibulárna disability blocker, EU EAA 2025 fail.

**Fix:** Globálne v root komponente:
```tsx
import { MotionConfig } from 'framer-motion';
<MotionConfig reducedMotion="user">{...}</MotionConfig>
```
Pre GSAP: `gsap.matchMedia({ '(prefers-reduced-motion: no-preference)': () => { /* animácie */ } })`.
